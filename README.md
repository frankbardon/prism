# Prism

Prism is a visualization library for [Pulse](https://github.com/frankbardon/pulse)
files. It takes a declarative chart spec plus a Pulse dataset and produces
publication-quality SVG, Canvas, and PDF output.

This repository is in early scaffolding. See `.planning/PROJECT.md` for the
full project brief, `.planning/ROADMAP.md` for the phase plan, and
`.planning/design/` for architecture notes.
